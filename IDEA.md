# FIRSTPRINT — The Definitive Idea Document

**"Has this been done before?"**

---

Every day, millions of people build something that already exists — and millions more trust something that's a copy. Both problems have the same root: no technology can tell you when two different-looking things are actually the same thing.

Search matches words. Plagiarism checkers match text. AI detectors guess who wrote it. None of them can see *structure* — the logic, the flow, the architecture underneath the surface. Two programs with identical logic but different variable names are invisible to every tool that exists. Two articles making the same argument with different words are invisible. Two products with the same architecture but different branding are invisible.

**Firstprint sees through surfaces to structure.**

Paste anything — code, a document, a design, a flow. Firstprint strips the surface and reveals the structural skeleton: the branching logic, the control flow, the dependency relationships, the architectural decisions. Then it searches the entire public internet for structural equivalents — things that *work the same way* regardless of how they look.

**If you're building:** See everything ever built that works like yours. In any language, any framework, any format. Stop reinventing what already exists. Start building on what's proven.

**If you're verifying:** Know instantly whether what you're looking at is original or a structural copy of something that came before. With cryptographic proof of who was first.

One input. One click. Two answers that change everything.

## The Engine

Structural fingerprinting via AST normalization, feature extraction across four layers (AST subtrees, control flow, dependencies, logic patterns), MinHash and SimHash for similarity-preserving comparison, rarity-weighted scoring that ignores common patterns and highlights rare structural overlaps, Ed25519 signed provenance ledger with Merkle chain integrity.

Built, operational, 55/55 tests passing, TypeScript and Python support. Live on GitHub.

## The Experience

Browser extension. VS Code extension. Web app. Paste or highlight. One click. Seconds. No registration required. The public internet is the starting registry — value on first use.

## The Flywheel

Every query builds the world's first structural registry. The registry makes every future query more accurate. The more people search, the more comprehensive the map of what structurally exists. The more comprehensive the map, the more valuable every query.

## The Moat

The rarity model requires neutral aggregation of structural patterns across competing platforms. No single AI company will share that data with rivals. Only Firstprint can build it.

## The Ceiling

Google indexed what the world *says*. Firstprint indexes what the world *does*. The first search engine for structure. The trust layer for the AI age.
